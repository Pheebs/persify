## Persify

A JavaScript library to convert English digits and numbers to Persian (Farsi) digits. It works in browser and Node.js.

[![Build Status][travis-image]][travis-url]

## Install

```bash
$ npm install persify
```

## Usage

```js
const persify = require('persify');

persify('Hello 123');
// => "Hello ۱۲۳"
```
