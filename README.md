## Persify

Convert English digits to Persian digits in browser and node.js

[![Build Status][travis-image]][travis-url]

## Install

```bash
$ npm install persify
```

## Usage

```js
const persify = require('persify')

persify('Hello 123')
// => "Hello ۱۲۳"
```
